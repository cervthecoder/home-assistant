# Jak nastavit home-assistant pro Solax

## Rest API
- Na webu (v mobilní aplikaci v5.5.0 nelze) solax cloud: **Service -> API -> OBTAIN TOKEN ID** 
- Dole se zobrazí API Adress:`https://eu.solaxcloud.com/proxyApp/proxy/api/getRealtimeInfo.do?tokenId={tokenId}&sn={sn}`
1. {tokenId} = Token ID, které jste obdrželi (mělo by být už doplněné)
2. {sn} = Registrační číslo vašeho inverteru (v žádném případě seiové číslo :DD): **Devices -> Inverter**
